API Documentation

Interactive API documentation available at:

  • /docs - Swagger UI
  • /redoc - ReDoc

API Endpoints

Base URL: /api/v1

  • GET /simulations - List available recordings
  • GET /simulations/{id}/metadata - Simulation metadata
  • GET /simulations/{id}/frames - Frame list
  • GET /simulations/{id}/frames/{fid}/state - Frame data
  • GET /simulations/{id}/frames/range?start={n}&count={n} - Bulk frames
  • GET /simulations/{id}/assets - Asset geometries
  • GET /simulations/{id}/energy - System energy
